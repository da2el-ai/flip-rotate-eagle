# FlipRotate

Eagleで選択した画像を一括で反転・回転処理できるプラグイン

## 機能

- **左右反転**: 選択した画像を左右反転
- **上下反転**: 選択した画像を上下反転
- **90度回転**: 選択した画像を時計回りに90度回転
- **180度回転**: 選択した画像を180度回転
- **270度回転**: 選択した画像を時計回りに270度回転（反時計回りに90度）

### 保存方式

- **新規保存**: 処理後の画像を新しいファイルとしてEagleに登録
  - 保存形式を選択可能（WEBP / PNG / JPEG / BMP）
  - JPEGの場合は品質も選択可能
- **上書き保存**: 元のファイルを処理後の画像で上書き
  - 元のファイル形式を維持

### 特徴

- 複数の画像を一括処理可能
- 前回の設定（保存方式・フォーマット・品質）を記憶

### 注意点

- PNGInfo、Exif情報は消えます。
- Eagleのメモ、タグなどは引き継げます。

## インストール方法

リポジトリのクローン、またはZIPのダウンロードで入手。

### リポジトリからクローン

```bash
git clone https://github.com/da2el-ai/flip-rotate-eagle.git
```

### ZIPをダウンロード

https://github.com/da2el-ai/flip-rotate-eagle/archive/refs/heads/main.zip

### Eagleにインポート

1. Eagleのメニューから **［プラグイン＞開発者オプション＞ローカルプロジェクトをインポート］** を選択
2. cloneまたは展開したフォルダを選択

## 使い方

1. Eagleで処理したい画像を選択
2. プラグインメニューから **「FlipRotate」** を選択
3. ダイアログが開くので保存方式を選択
   - **新規保存**: 新しいファイルとして保存（フォーマット選択可能）
   - **上書き保存**: 元のファイルを上書き（元の形式を維持）
4. 処理ボタンをクリックすると実行される
   - 左右反転 / 上下反転 / 90度回転 / 180度回転 / 270度回転

## 対応フォーマット

### 入力

- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)
- BMP (.bmp)
- GIF (.gif)

### 出力（新規保存時）

- **WebP**: 最高品質（可逆圧縮）
- **PNG**: 可逆圧縮
- **JPEG**: 品質選択可能（60% / 80% / 90% / 95%）
- **BMP**: 無圧縮
